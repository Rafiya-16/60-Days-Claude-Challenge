# Day 42 – Personal Financial Command Center

## Overview
Today's challenge was to build a complete **Personal Financial Command Center** using Claude AI. The application helps students manage their finances through budgeting, expense tracking, savings planning, investment simulations, and AI-powered financial insights—all within a modern interactive dashboard.

---

## Objective
Create a fully functional financial management web application that enables users to:

- Track monthly income
- Monitor expenses
- Build category-wise budgets
- Manage savings goals
- Track subscriptions
- Calculate SIP investments
- Run financial what-if simulations
- Receive AI-generated financial recommendations
- Monitor overall financial health
