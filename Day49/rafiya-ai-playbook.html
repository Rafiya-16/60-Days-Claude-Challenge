What is this?

This is your personal library of reusable AI prompt systems — not one-off prompts, but editable templates you can reuse across Claude, ChatGPT, and other tools. Instead of retyping instructions every time, pick a saved workflow, fill in a few variables, and copy a ready-to-use prompt.

Workflow Library — ready-made prompt templates organized by what you do (coding, learning, job search).
Prompt Builder — assemble a custom prompt from labeled building blocks, with a live preview.
Loop Builder — turn any prompt into a self-checking, self-improving instruction set.

Loop Builder

Turns any normal prompt into an autonomous looping prompt — one where the AI checks its own work against criteria you set, improves it, and repeats until it meets a stop condition (like "all tests pass" or "3 rounds max"). Useful for things like iterating on code until requirements are met.
