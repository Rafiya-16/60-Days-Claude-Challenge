# Day 6 – Resume Optimizer

## Objective

Learn how to use Claude as an ATS Resume Optimizer to analyze, improve, and rewrite a resume for better recruiter visibility and ATS compatibility.

---

## Tools Used

* Claude AI
* GitHub

---

## Original Resume Analysis

### ATS Score Comparison

| Resume Version   | Score    |
| ---------------- | -------- |
| Original Resume  | 41 / 100 |
| Optimized Resume | 78 / 100 |

### Key Improvements Suggested by Claude

* Replaced the generic Career Objective with a keyword-rich Professional Summary.
* Standardized section headings for better ATS parsing.
* Improved project descriptions using stronger action verbs.
* Organized technical skills into structured categories.
* Simplified contact information formatting.
* Removed vague and repetitive content.
* Added industry-relevant keywords for Software Engineering and Front-End Development roles.
* Improved readability and recruiter scanning efficiency.

---

## Optimized Resume Highlights

### Professional Summary

A concise summary was added to highlight:

* Computer Science background
* Software Development interests
* Front-End Development skills
* Git/GitHub experience
* AI-related career goals

### Technical Skills

* C
* C++
* Java
* JavaScript (Learning)
* HTML5
* CSS3
* SQL
* Git
* GitHub
* VS Code

### Project Improvements

Project descriptions were rewritten using action-oriented language such as:

* Designed
* Developed
* Implemented
* Managed
* Organized
* Applied

This made the projects more impactful and ATS-friendly.

## Key Learnings

1. ATS systems rely heavily on keywords.
2. Professional summaries perform better than generic objectives.
3. Structured section headings improve ATS parsing.
4. Strong action verbs increase resume impact.
5. Project descriptions should focus on contributions and outcomes.
6. Clean formatting improves recruiter readability.
7. GitHub projects can strengthen a student resume even without professional experience.
8. Continuous improvement of skills and certifications can significantly increase ATS scores.

---

## Outcome

Successfully used Claude to analyze and optimize a resume.

The resume ATS score improved from **41/100 to 78/100**, resulting in a more professional, recruiter-friendly, and ATS-compatible resume suitable for Software Engineering, Front-End Development, and Full-Stack Development internship opportunities.
